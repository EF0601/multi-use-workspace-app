Help

Basic Operations

  • Use the buttons to input numbers and operators
  • Use the C button to clear the screen
  • Use the Del button to delete the last character
  • Use the = button to calculate the result

Memory Operations

  • Use the M+ button to add the current value to memory
  • Use the M- button to subtract the current value from memory
  • Use the MR button to recall the value stored in memory
  • Use the MC button to clear the value stored in memory

Advanced Operations

  • Use the pi button to input the value of pi
  • Use the x! button to calculate the factorial of a number
  • Use the x^2 button to calculate the square of a number
  • Use the 1/x button to calculate the reciprocal of a number
  • Use the x^3 button to calculate the cube of a number
  • Use the |x| button to calculate the absolute value of a number
  • Use the sqrt(x) button to calculate the square root of a number
  • Use the y sqrt(x) button to calculate the y-th root of a number
  • Use the x^y button to calculate the exponentiation of a number
  • Use the 3 sqrt(x) button to calculate the cube root of a number
